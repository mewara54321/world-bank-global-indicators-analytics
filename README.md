# 🌍 World Bank Global Indicators Analytics

## 📌 Project Overview

This project aims to build an **end-to-end data analytics pipeline** using the **World Bank Open Data API** to analyze global development indicators.

The project will collect, process, and visualize key indicators such as:

* Economic growth
* Labour market conditions
* Trade indicators
* Poverty and inequality
* Health indicators
* Environmental indicators
* Technology adoption

The final outcome will be an **interactive Power BI dashboard** that allows users to explore global development trends across countries and years.

---

## 🎯 Project Objectives

The main goals of this project are:

* Extract global indicator data from the **World Bank API**
* Build a **Python data pipeline** for data extraction and transformation
* Create a **clean analytics dataset**
* Design an **interactive Power BI dashboard**
* Demonstrate a **complete data analytics workflow**

## 📚 Data Source

The data used in this project is retrieved from the **World Bank Open Data API**, which provides access to global development indicators across countries and years.

World Bank API Documentation:
https://datahelpdesk.worldbank.org/knowledgebase/articles/889386-developer-information-overview

Example API Endpoint:
https://api.worldbank.org/v2/country?format=json

API Directory Listing:
https://publicapi.dev/world-bank-api

The API provides information on:

* Economic indicators (GDP, trade)
* Labour market indicators
* Poverty and inequality
* Health statistics
* Environmental indicators
* Technology adoption
